BerryCam Express

Control your Raspberry Pi camera module from any device.

Download project as: .zip file tar.gz file

BerryCam just got better.

BerryCam Express is a complete rewrite in every sense of the word.

Using Node and Knockout JavaScript libraries, BerryCam Express allows you to access the same interface as the original BerryCam app for iOS, using fully opensource components directly from your Raspberry Pi.

The result is a super modular JavaScript web-app that you run directly on the Pi, and there's no limits to the devices you can connect. BerryCam Now works with any device on any platform that uses a web browser and has network connectivity including:

Android iOS Windows Mobile Linux

A great way to take images.

Using BerryCam Express is easy. Control all your camera settings from within the app, and quickly preview and share your images. It removes the need to fiddle around with the command line interface (although we recommend that you give it a try) and allows you to use the camera as a serious image capture tool.

We've worked hard to maximise what you can get out of the camera. From beautiful, crisp still images with all the built-in image effects, to creating stunning timelapse sequences (coming soon).

Unlock your creativity using from the simple interface on the device of your choosing.

A great way to learn about code.

BerryCam Express is also great way to learn more about developing software using current technologies like JavaScript, Node, HTML5, LESS. After all the Raspberry Pi is about learning!

The app itself is written in a way that will help you get to grips with Object Oriented Programming (OOP), MVVM, MVC and Test Driven Development (TDD). So BerryCam Express is a great project for those looking to brush up their knowledge in these areas or even as a supporting part of a curriculum or study project of your own.

We've supplied all the source code that will allow you to customise the app to your own requirements. We really cannot wait to see what people out there will do with the app!
