BerryCam Express

Control your Raspberry Pi camera module from any device.

Download project as: .zip file tar.gz file

BerryCam has just got better.

BerryCam Express is a complete rewrite in every sense of the word. Using Node and Knockout JavaScript libraries, BerryCam Express allows you to access the same interface as the original BerryCam app for iOS directly from your Pi.

It's pretty much a super modular JavaScript web-app that you run directly on the Pi, so there's no limits to the devices you can connect. BerryCam Now works with any device on any platform that uses a web browser and has network connectivity including:

Android iOS Windows Mobile Linux
